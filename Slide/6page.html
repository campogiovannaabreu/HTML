Quais são as 3 formas de
"ligar" o CSS no HTML?

Inline

externa

interna


Neste método, as regras CSS são aplicadas diretamente a
um elemento HTML específico usando o atributo style.


Este é o método mais comum e recomendado para projetos maiores. Os estilos são escritos em um arquivo

separado com a extensão .css (por exemplo, styles.css) e, em seguida, vinculados ao documento

HTML usando a tag link dentro da seção head. Permite centralizar todos os estilos em um só lugar,

facilitando a manutenção e a aplicação de um visual consistente em múltiplas páginas do site.


O CSS é colocado dentro da própria página HTML, dentro da tag style no head.
