Seletor TAG


Seleciona todos os elementos de um documento HTML

Aplicará o estilo a todas as tags p e h1

Seletor ID


Esse seletor seleciona um único elemento com um

identificador, É usado com # e em seguida o nome do ID

e assim será selecionado apenas o elemento

Seletor CLASSE


Ele seleciona todos os elementos que possuem um

determinado valor de atributo de classe. Com ponto (.) e em

seguida o nome da classe, aplicando-se a todos os elemento com (.)

Seletor UNIVERSAL


O (*) seleciona todos os elementos do html sem exceção, usado para definir as margens e preenchimento padrão.
