Por que o W3C cria regras para a web?

O W3C cria regras, ou padrões (como

HTML, CSS e acessibilidade), para garantir que a web funcione de forma

consistente e acessível para todos, independentemente do dispositivo

ou navegador utilizado. Essas regras promovem a interoperabilidade, o

que significa que um site construído seguindo os padrões funcionará

da mesma maneira no Chrome, Firefox, Safari, e outros navegadores.

O W3C (World Wide Web Consortium) é um consórcio internacional que atua como o principal órgão de padronização da web.
